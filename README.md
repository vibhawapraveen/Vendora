# Vendora 🛒

**Vendora** is a pre-built web platform where sellers can register, create a store, and manage both physical and online sales in one place—no coding or setup required.

## Features

* 🔐 Vendor authentication
* 💳 Stripe Connect integration (standard accounts)
* 📦 Order and payout tracking
* 🧾 Lightweight PHP MVC structure

## Tech Stack

* HTML
* CSS
* JS
* PHP (Vanilla MVC)
* MySQL
* Stripe Connect (Standard)
